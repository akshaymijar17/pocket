Mozilla killed my beloved Pocket App sometime last year, late 2025. I was a loyal user from 2014 and it was one of the apps that I used the most on my phone as I would drop copious amounts of links through the weeks, things that I found interesting, things I wanted to archive for later and over a period of time this list became a thing, this list became a directory of items that expanded how I think. I did not like the other alternatives out there such as Instapaper etc. We did not want to pay for like a simple link management service like this. This is my attempt to build a cross-platform clone of Pocket. However, instead of listing it as an app on the App Store, I am using a simple Telegram bot to where I can dump the links that I want to save for later telegram bot then places This is the links in my application on Firebase.
